# 基于深度学习的中药智能识别系统 python269

## 项目概述

本项目是一个基于深度学习的中药智能识别系统，旨在通过现代技术提高中药材的识别准确率，以促进中药材研究和应用的发展。

## 技术栈

### 后端

- **框架**: 使用 Python 的 Flask 框架进行后端开发。
- **数据库**: 采用 SQLite 数据库管理系统。
- **训练与预测**: 使用 DenseNet121 模型进行图像识别，结合 TensorFlow 算法进行深度学习训练、评估和预测。

### 前端

- **技术**: 使用 HTML、CSS 和 JavaScript 进行前端开发。
- **框架**: 采用了如 jQuery 等主流的前端框架和组件。

### 移动端

- **开发环境**: 使用 Android Studio 进行开发。
- **编程语言**: 采用 Kotlin 语言编写移动应用。

## 功能模块

- **Web 端**: 提供通过网页浏览器访问的中药材识别服务。
- **移动端**: 开发了对应的移动应用程序，以方便用户在移动设备上使用。
- **智能识别**: 能够准确识别61种中药材，识别准确率达到98%以上。

## 系统角色

- **用户**: 使用系统的主体，可以访问web端和移动端应用。
- **管理员**: 负责系统的维护和管理。

## 运行环境

- **后端**: 推荐在 Python 环境下运行，具体版本依赖见项目文档。
- **前端**: 任何支持现代Web标准的浏览器。
- **移动端**: Android 操作系统。
